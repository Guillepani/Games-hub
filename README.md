# Games Hub

Proyecto 5 del bootcamp desarrollado con Vite y JavaScript vanilla.

## Descripción

Aplicación web compuesta por 3 juegos:

- Tres en raya
- Snake
- Tower Builder

El objetivo del proyecto es trabajar la lógica de programación, la maquetación, la componentización y el uso de localStorage para conservar la puntuación.

## Tecnologías

- HTML
- CSS
- JavaScript
- Vite

## Requisitos del proyecto

- Web full responsive
- Buen diseño y maquetación
- 3 secciones accesibles con un juego en cada una
- Un tres en raya obligatorio
- Juegos sin bugs
- Código bien componentizado
- Persistencia de puntuación con localStorage
